# 🖥️ PC to PC Network using Crossover Cable

## 🎯 Objectives
- Establish direct communication between two PCs
- Verify data transmission without networking devices

## 🛠️ Software
- Cisco Packet Tracer

## ⚙️ Procedure
1. Two PCs were added in the workspace  
2. A crossover cable was used to connect FastEthernet ports  
3. IPs were assigned in the same subnet  
4. Connectivity was tested using `ping`  

## 🌐 IP Configuration Example
| Device | IP Address     | Subnet Mask     |
|--------|---------------|-----------------|
| PC1    | 192.168.1.1   | 255.255.255.0   |
| PC2    | 192.168.1.2   | 255.255.255.0   |

## 📌 Key Insights
- Direct communication was established successfully  
- No hub/switch was required  
- Same network IP was mandatory  
- Ping confirmed connectivity  

## 📚 Concept
Peer-to-peer communication using crossover cable without intermediate devices.
